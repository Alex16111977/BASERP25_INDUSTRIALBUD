Контроль кодов УКТВЭД товаров

Отчет предназначен для контроля корректности отражения учета операций с товарами с указанием кода для НН и номера ГТД. При помощи отчета можно оперативно выявлять случаи неправильно указания кодов для НН и номеров ГТД при поступлении и отгрузке товаров.

Особенности отчета:

  • Для анализа доступны количественные данные поступившего и отгруженного товаров;
  • Предусмотрена детализация по организации, номенклатуре.
