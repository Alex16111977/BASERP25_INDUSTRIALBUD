Дані для бази розподілу витрат

Звіт призначений для інформування про наявність даних для побудови бази розподілу витрат.

Особливості звіту:

  • Наявність даних для бази не означає, що база розподілу гарантовано буде побудована, тобто при побудові бази автоматично програмно накладається ряд фільтрів на дані і т. д.
  • У деяких випадках звіт може не показувати дані для побудови бази, але це не означає, що витрати в результаті не розподіляться. Наприклад, якщо використовується база розподілу Сума витрат на оплату праці, то дані цієї бази з'являться в процесі виконання операцій партіонного обліку по віднесенню трудовитрат на випуск продукції.
