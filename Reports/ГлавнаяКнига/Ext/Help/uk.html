Звіт "Головна книга"

Звіт дозволяє за кожним рахунком (субрахунком) вивести інформацію про сальдо початкове і кінцеве періоду, оборотах рахунку з іншими рахунками (субрахунками) за обраний період часу. Звіт формується за даними бухгалтерського обліку.

Побудова звіту виконується по кнопці Сформувати.

Формування більш компактного виду звіту з прихованим заголовком виконується по кнопці Заголовок.

Параметри формування звіту можна задати по кнопці Настройка:

  • У полі Період можна задати період формування звіту: місяць, рік тощо.
  • Для деталізації даних по субрахунках або по субрахунках кор.рахунків потрібно встановити прапорець По субрахунках або По субрахунках кор.рахунків.
  • Для виведення розгорнутого сальдо потрібно встановити прапорець Розгорнуте сальдо. Додаткові параметри можна задати на закладці Розгорнуте сальдо, наприклад, вказати по субрахунках чи по аналітиці рахунків має виводиться у звіт розгорнуте сальдо. Кнопку По умовчанню можна заповнити настройки, які встановлені у звіті по умовчанню, і при необхідності їх скорегувати.
  • Під час друку можна вивести дані кожного рахунку (субрахунку) на окремий аркуш. Для цього потрібно встановити прапорець Розбити по листам.
