Умови купівель

Звіт призначений для порівняння цінових умов щодо купівлі товарів зареєстрованих в довіднику Оферти постачальникам.
Звіт доступний при включеній функціональної опції Замовлення постачальникам в розділі НДІ та адміністрування- Купівлі-Замовлення постачальникам.

Особливості звіту:

  • Аналіз цінових умов в рамках укладених оферт по номенклатурі/характеристиці, партнеру, сегменту;
  • Аналіз за офертами доступним тільки для купівлі (прапор Показувати оферти доступні для купівлі);
  • Аналіз за офертами доступним тільки для продажу (прапор Показувати оферти доступні для продажу клієнтам);
  • Аналіз мінімальних цін за обраними позиціями номенклатури (прапор Показувати тільки рекомендовані).


Для можливості відбору за мінімальною ціною в звіті повинні бути заповнені поля Валюта і Дата звіту.
