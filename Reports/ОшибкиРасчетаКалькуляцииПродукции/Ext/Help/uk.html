Помилки розрахунку калькуляції продукції

Звіт призначений для виявлення помилок при розрахунку планової вартості продукції.
Відображає:

  • перелік матеріалів необхідних для виробництва продукції для яких не встановлена планова ціна;
  • трудовитрати, необхідні для виробництва продукції для яких не встановлені розцінки.
