Призначена для вибору типу предмета в документах взаємодій.

Для заповнення поля Предмет спочатку виберіть зі списку тип предмета. Після цього відкриється список обраних об'єктів програми.

За замовчуванням у списку відображаються типи документів взаємодій.

Для відображення інших типів об'єктів програми увімкніть прапорець Не відображати взаємодії.
