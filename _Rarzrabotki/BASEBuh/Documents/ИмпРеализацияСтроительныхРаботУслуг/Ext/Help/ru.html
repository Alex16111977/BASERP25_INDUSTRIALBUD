Документ "Реализация строительных работ и услуг".

Для отражения факта передачи заказчику выполненных строительно-монтажных работ, которые выполненных своими силами и силами субподрядчиков, предназначен документ «Реализация строительных работ, услуг».

В шапке документа указываются

  • организация,
  • номер и дата составления документа,
  • заказчик, договор,
  • объект строительства (номенклатурная группа),
  • подразделение организации, выполнявшее работы.

В документе также предусмотрены закладки:

  • собственные СМР,
  • субподрядные СМР,
  • счета учёта расчётов,
  • признанный доход БУ,
  • признанный доход НУ,
  • списание ТМЦ заказчика,
  • дополнительно.

 

На закладке «Собственные СМР» в табличной части перечисляются работы, которые выполнены с указанием количества, цены, суммы, суммы НДС, суммы всего.

Если есть ранее созданный документ «Акт приема-передачи КБ-2в (факт), то кнопкой «Заполнить» указанные выше реквизиты табличной части загружаются автоматически из документа «Акт приема-передачи КБ-2в (факт).

В таблице можно менять количество, цену за единицу, сумму, НДС, сумму с НДС, счета учета реализации.

На закладке «Субподрядные СМР» в табличной части из справочника «Контрагенты» выбирается субподрядчик.

Далее выбирается договор, проставляется сумма субподрядных работ, сумма НДС, сумма всего.

По кнопке «Заполнить» указанные данные могут заполниться автоматически по данным проведенного ранее документа «Приемка строительных работ, услуг».

В таблице можно менять количество, цену за единицу, сумму, НДС, сумму с НДС, счета учета реализации.

На закладке «Признанный доход БУ» заполняются показатели по сумме передачи выполненных работ по СМР, а именно:

  • сумма переданных работ без НДС,
  • сумма НДС,
  • сумма выполненных работ с НДС.

 

Данные для показателя «Сумма переданных работ без НДС» выбираются и суммируются из закладок: «Собственные СМР» и «Субподрядные СМР».

 

Далее вручную заполняется показатель фактических расходов по данным бухгалтерского учёта.

 

По кнопке «Рассчитать» автоматически заполняются следующие показатели:

  • Показатель расходов по смете (данные выбираются из документов «Строительный контракт» и «Изменение суммы строительного контракта» (в части изменений) на дату документа),
  • Сумма признанных доходов прошлых периодов (выбирается по накопленным суммам по дебету счета БКБ в разрезе договора, контрагента и объекта строительства за период, предшествующий заполняемому документу),
  • Сумма признанного дохода по документу (определяется расчетным путем согласно действующей методике),
  • Полная сумма строительного контракта (данные выбираются из документов «Строительный контракт» и «Изменение суммы строительного контракта» (в части изменений) на дату документа).

 

Методика расчёта суммы признанного дохода по данным бухгалтерского учёта:

 

1. Наиболее привычным для строительных предприятий является установление степени завершенности работ для определения дохода по методу измерения и оценки выполненной работы (метод № 1), который положен в основу ведения журнала (КБ-6) с последующим составлением форм КБ-2в и КБ-3. Сумма признан­ного дохода, рассчитываемого по вышеупомянутому методу, совпадает с суммой, указанной в акте.

Для целей бухгалтерского учета сумма признанного дохода может совпадать с суммой промежуточных счетов только в случае, если степень завершенности ра­бот по строительному контракту будет определяться путем измерения и оценки выполненной работы на основании акта выполненных работ (по методу № 1).

При условии, что сумма признанного дохода равна сумме промежуточного счета, корреспонденция счетов бухгалтерского учета расчетов с заказчиками по выполненным работам (этапам работ) будут иметь следующий вид:

 

Д-т 361 К-т 703 - на сумму акта (промежуточного счета);

Д-т 703 К-т 6432  на сумму НДС (по первому событию);

Д-т 239 (незавершенные строительные контракты) К-т 238 (промежуточные счета) - на сумму акта без НДС;

Д-т ВД К-т ВД - на сумму валового дохода (сумма акта без НДС для

не долгосрочных СК и расчетную сумму ВД для долгосрочных СК);

К-т БКБ - на сумму акта;

К-т БКП - на сумму признанного ВД;

К-т БКН - на сумму учтенного НДС.

К-т БКВ - на сумму учтенных фактических показателей расходов по смете (по документу).

 

 

2. Для определения дохода, используя метод № 2 (соотношения фактических рас­ходов с начала выполнения строительного контракта до даты баланса и ожидае­мой (сметной) суммой общих расходов по контракту), необходимо:

 

2.1) определить коэффициент соотношения (К) фактически признанных расходов (Рфакт) и плановых расходов (Рплан) по строительному контракту:

К = Рфакт / Рплан;

Рфакт (графа табличной части «Показатель фактических расходов в бухгалтерском учёте») заполняется пользователем вручную .

Рплан (графа табличной части «Показатель по смете») заполняется из документа "Договоры контрагентов" реквизит "Общая сумма расходов по смете" закладка "Строительные контракты".

 

2.2) определить признанный доход (ПД) с начала строительства, умножив плановый доход (договорную цену - общую сумму с НДС по договору СК) (ДЦ) на коэффици­ент соотношения (К):

ПД = ДЦ х К;

ДЦ заполняется из документа "Договор" реквизит параметров договора "Сумма (с НДС)".

 

2.3) Определить непосредственно размер признанного дохода за определенный этап как разницу между суммой признанного дохода (ПД) нарастающим итогом и суммой признанного дохода предыдущих этапов.

Сумма признанного дохода предыдущих этапов, участвующая в расчете - это кредитовые обороты по счету БКБ  за весь период от начала выполнения СК.

При методе № 2 сумма счета, выставленного заказчику, не будет совпадать с суммой признанного дохода по бухгалтерскому учету.

В ПСБУ 18 дано определение такого показателя, как промежуточные счета, ко­торый будет играть важную роль в порядке отражения доходов на счетах бухгал­терского учета.

      Промежуточные счета - это счета за выполненные работы по строительному контракту, переданные заказчику для оплаты. В практике, промежуточные счета выставляются заказчику для оплаты, исходя из подписанных актов выполненных работ, оказанных услуг.

     Согласно п. 18 ПСБУ 18 сумма промежуточных счетов отражается согласно Плану счетов одновременно по дебету субсчета 238 "Незавершенные строительные контракты" и кредиту субсчета 239 "Промежуточные счета" счета 23 "Производство". После завершения строительного контракта сальдо на указанных субсчетах списывается обратной корреспонденцией субсчетов.

 

3. Для исчисления дохода за определенный период (этап), используя метод № 3 (соотношения объема завершенной части работ и их общего объема по строительному контракту в натуральном измерении), выполняются следующие действия:

 

3.1)  Определяется коэффициент соотношения (К) объема завершенной части работ (ОЗР) и общего объема работ (ООР) по строительному контракту:

К = ОЗР / ООР

В качестве показателя объема работ могут быть использованы такие величины, как количество человеко-часов, фактически отработанных и планируемых, количество машино-часов, отработанных и планируемых, кубометры кирпичной кладки, квадратные метры обустройства кровли и т.п. Однако достоверный результат при использовании данного метода может быть получен при условии однородности работ на протяжении всех этапов строительства.

Реквизит ОЗР (графа табличной части «Показатель фактических расходов в бух­галтерском учёте»)) вводится пользователем вручную в диалоге документа.

Реквизит ООР (графа табличной части «Показатель по смете») заполняется из документа "Договоры контрагентов" реквизит "Общий объем работ по смете в на­туральных измерителях". При этом открывается для выбора вида натурального измерителя справочник "Классификатор единиц измерения" с помощью которого пользователь фиксирует выбранный натуральный измеритель сметы, который бу­дет учитываться в расчетах сумм признанного дохода по мере выполнения строи­тельного контракта.

 

3.2) определяется признанный доход (ПД) с начала строительства путем умножения планового дохода (договорной цены (ДЦ)) по строительному контракту на коэффициент соотношения (К):

ПД = ДЦ х К

Реквизит ДЦ аналогичен описанному в методе № 2.

 

3.3) определяется непосредственно размер признанного дохода за определенный этап как разница между суммой признанного дохода (ПД) нарастающим итогом и суммой признанного дохода предыдущих этапов (опреде­ляется аналогично в методе№2).

Проводки, формируемые документом при использовании метода определения

дохода № 2 и № 3, следующие:

Д-т 361 К-т 703 - на сумму признанного дохода;

Д-т 703 К-т 6432  на сумму НДС промежуточного счета;

Д-т 239 (НСК) К-т 238 (ПС) - на сумму промежуточного счета без НДС;

Д-т 361 К-т 239 (НСК) - на сумму разницы между суммой промежуточного

счета и суммой признанного по бухгалтерскому учету дохода;

Д-т ВД К-т ВД - на сумму валового дохода (сумма акта без НДС для недолгосрочных СК и расчетную сумму ВД для долгосрочных СК);

К-т БКБ - на сумму акта;

К-т БКП - на сумму признанного ВД;

К-т БКН - на сумму учтенного НДС.

      К-т БКВ - на сумму учтенных фактических показателей расходов по смете (по документу).

 

На закладке «Признанный доход НУ» вручную заполняется показатель фактических расходов по данным налогового учёта.

 

По кнопке «Рассчитать» автоматически заполняются следующие показатели:

  • Показатель расходов по смете (данные выбираются из документов «Строительный контракт» и «Изменение суммы строительного контракта» (в части изменений) на дату документа),
  • Сумма признанных доходов прошлых периодов по налоговому учёту (выбирается по накопленным суммам по дебету счета БКП в разрезе договора, контрагента и объекта строительства за период, предшествующий заполняемому документу),
  • Сумма признанного дохода по налоговому учёту по документу (определяется расчетным путем согласно действующей методике),
  • Полная сумма строительного контракта по налоговому учёту (данные выбираются из документов «Строительный контракт» и «Изменение суммы строительного контракта» (в части изменений) на дату документа),
  • НДС с учетом признанного ВД.

 

Методология расчёта признанного валового дохода и НДС по долгосрочным строительным контрактам.

 

     Между подрядной организацией и заказчиком заключаются договора, которые носят долгосрочный характер. Под Долгосрочным договором (контрактом) согласно пп. 7.10.2 ст. 7 Закона «О налогообложении прибыли предприятий» следует понимать любой договор на изготовление, строительство, установку или монтаж материальных ценностей, которые будут входить в состав основных фондов за­казчика или составных частей таких основных фондов и т. п. при условии, что та­кой контракт не планируется завершить ранее чем через 9 месяцев с момента осуществления первых затрат или получения предоплаты.

     Пунктом 7.10 ст. 7 Закона о налогообложении прибыли определен порядок налогообложения доходов от исполнения долгосрочных договорных обязательств.

Особенностью налогового учета операций по выполнению долгосрочных контрак­тов является то, что налогоплательщик имеет право сам выбирать особый поря­док налогообложения результатов деятельности, которая осуществляется по дол­госрочным договорам, или же применять общие правила для налогообложения.

     При применении особого порядка налогообложения результатов деятельности, осуществляемой по долгосрочным контрактам, исполнитель самостоятельно оп­ределяет доходы в течение отчетного периода для целей налогообложения в связи с выполнением этого договора, используя оценочный коэффициент выпол­нения такого долгосрочного контракта.

     В конфигурации  реализован данный подход для определения суммы валовых доходов по долгосрочному строительному контракту. Для идентификации долго­срочных СК в справочнике "Договоры контрагентов" на закладке Строительные кон­тракты предусмотрен флаг "Договор является долгосрочным для налогового учета".

    Если данный флаг не включен, то определение валовых доходов по строитель­ному контракту осуществляется в обычном порядке (по первому событию).

    

Если флаг включен, то порядок определения суммы ВД следующий:

 

     Валовой доход исполнителя долгосрочных контрактов определяется по фор­муле:

 

ВД = Цдог х Коц, где

ВД - валовой доход исполнителя в отчетном периоде;

Цдог - общедоговорная цена долгосрочного контракта.

Цдог - учитывается из значения реквизита "Сумма (с НДС)" документа "Договоры контрагентов".

 

     Оценочный коэффициент выполнения долгосрочного договора определяется следующим образом:

 

Коц = Зфактпер / Зплан, где

Коц - оценочный коэффициент;

Зфактпер - фактические затраты отчетного периода, .

Зплан - общие затраты, которые планируются исполнителем при выполнении долгосрочного договора (контракта), определяются из реквизитов, заполненных пользователем в соответствии со сметой, справочника "Договоры контрагентов", закладка "Строительные контракты":

  • "общая сумма расходов по смете" для метода определения доходов "соотно­шения фактических расходов к сметным расходам";
  • "общая сумма расходов по смете" для метода определения доходов "измере­ния и оценки выполненных работ";
  • "общий объем работ по смете в натуральных измерителях" для метода определения доходов "соотношение объема выполненных работ к смет­ному в натуральных измерителях".

 

     Рассчитанная таким образом сумма валового дохода по документу отобража­ется при нажатии кнопки "Рассчитать" на закладке "Признанный доход НУ". Реквизит "Сумма учтенного ВД прошлых периодов" соответ­ствует сумме, накопленной по кредиту счета БКП. После проведения документа расчетное значение ВД по документу  отразится также по кредиту счету БКП.

     Если подрядная организация перешла на применение особого порядка налого­обложения результатов деятельности по долгосрочным договорам, то для учета НДС необходимо руководствоваться следующим.

     В соответствии с пп.7.3.7 п. 7.3 ст. 7 Закона об НДС датой возникновения нало­говых обязательств по НДС исполнителя долгосрочных договоров является дата увеличения валового дохода исполнителя долгосрочного контракта.

     Корректировка налоговых обязательств по НДС по итогам работы предприятия отражается в налоговой декларации по НДС и в расчете корректировки сумм НДС в месяце завершения долгосрочного контракта (приложение 2 к налоговой на­кладной).

     В соответствии с Порядком № 165 налоговая накладная составляется в двух экземплярах (оригинал и копия) в момент возникновения налоговых обязательств продавца (п. 6 Порядка № 165).

     Законом о налогообложении прибыли установлено, что для целей налогообло­жения суммы авансов, предоставленных исполнителю долгосрочного контракта, относятся в состав валового дохода такого исполнителя исключительно в преде­лах сумм, определенных согласно пп. 7.10.5 п. 7.10 ст.7 Закона о налогообложе­нии прибыли, т.е. как произведение договорной цены долгосрочного контракта и оценочного коэффициента исполнителя долгосрочного контракта.         Исходя из рассчитанной суммы ВД, возникают и налоговые обязательства по   НДС.

     Таким образом, налоговая накладная при выполнении долгосрочных договоров (контрактов) составляется исходя из расчетной величины валового дохода пред­приятия, а не из фактической суммы поступившего аванса.

      В конфигурации проводка по НДС из документа "Реализация строительных работ, услуг" (Д-т 703 К-т 6432) записывается на сумму НДС по промежуточному счету (фактическая сумма передачи).

     На закладке "Признанный доход НУ" имеется флаг "Учитывать корректи­ровку по НДС", при включенном флаге сумма НДС, рассчитанная от суммы ВД (информационно выводится по реквизиту "НДС с учетом ВД"), сравнивается с суммой НДС на закладке "Признанный доход БУ (реквизит "Сумма НДС") и раз­ница (со знаком плюс или минус) уменьшает или увеличивает налоговые обяза­тельства по НДС (Д-т 6432 К-т 6412). 

Данная прооводка проводиться только с учетом заполнения приложения №2 к налоговой накладной.   По этому документу данная проводка проводиться не будет.

     Если флаг не установлен, то корректировка отражаться не будет, а в обяза­тельство пойдет сумма НДС по промежуточному счету.

     Если корректировка суммы НДС учтена, то, воспользовавшись функцией "Ввод на основании", к документу "Налоговая накладная", введенному на основании до­кумента «Реализация строительных работ, услуг", следует в обязательном порядке ввести Приложение № 2 на сумму проведенной корректировки.

     По забалансовому учету (счет "БКН") по дебету отражается сумма общая НДС по всему строительному контракту, а с кредита списывается сумма НДС по доку­менту, которая учтена в обязательствах по НДС отчетного периода (или с коррек­тировками, или без них).

     При завершении СК пользователь имеет возможность провести одну общую корректировку налоговых обязательств по НДС в месяце завершения СК (при этом счет БКН закроется в ноль).

 

На закладке "Списание ТМЦ заказчика" можно списать материалы , переданные заказчиком подрядчику, в обеспечение процесса выполнения СМР.

По кнопке "Заполнить" предлагается осуществить три варианта отбора для заполнения:

- Заполнить из поступления;

- Добавить из поступления;

- Заполнить по остаткам (выбираются остатки по субсчетам забалансового счета 021 "Оборудование принятое для монтажа").

При списании материалов заказчика документ делает проводку: Кт 021 (в разрезе субсчетов).

 

На закладке «Дополнительно» -  вводятся данные об инвесторе, подрядчике, заказчике, период выполнения работ.

 

 В документе формируется печатная форма «Акт приема-передачи выполненных работ» произвольной формы и Унифицированная форма справки о стоимости строительных работ №КБ-3.

Отчеты формируются в той валюте, которая используется в документе.
