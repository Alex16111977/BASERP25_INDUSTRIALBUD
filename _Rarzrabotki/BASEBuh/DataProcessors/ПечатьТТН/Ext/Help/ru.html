Обработка "Печать товарно-транспортной накладной"

Данная обработка предназначена для печати документов «Реализация товаров и услуг» и «Перемещение товаров» в виде печатной формы товарно-транспортной накладной.

Выбор формы производится в форме документов при нажатии на кнопку Печать и выбора пункта меню 1-ТН (Товарно-транспортная накладная). При этом открывается диалоговое окно, в котором необходимо заполнить дополнительные реквизиты, которые необходимы для печати товарно-транспортной накладной:

  • Номер ТТН - по умолчанию заполняется числовой номер документа Реализация товаров и услуг (Перемещение товаров) с префиксом "Р"("П"), может редактироваться;
  • Место составления документа - заполняется из документа Реализация товаров и услуг или из настройки пользователя Основное место составления документа (Главное - Персональные настройки);
  • Пункт погрузки - заполняется из местонахождения Склада-отправителя или фактического адреса Организации;
  • Пункт разгрузки - заполняется из Адреса доставки накладной или фактического адреса Контрагента (Реализация товаров и услуг), местонахождения Склада-получателя (Перемещение товаров);
  • Масса брутто груза - заполняется вручную;
  • Масса общая с грузом - рассчитывается как Масса брутто груза + Масса по техпаспорту транспортного средства и может корректироваться вручную;
  • Информация про автомобильного перевозчика, Вид перевозки - заполняются вручную;
  • Информация о водителе (экспедиторе) - заполняется вручную;
  • Информация о транспортном средстве - заполняется вручную. Масса по техпаспорту не выводится на печать и необязательна для заполнения; используется для расчета Массы общей с грузом.;

Заполненные в разделе "Сведения о доставке" реквизиты можно сохранить по кнопке Сохранить параметры доставки для использования в других ТТН.

По кнопке Выбрать параметры доставки выводится список сохраненных ранее параметров (наборов реквизитов ТТН). При выборе набора параметров заполняются реквизиты печати ТТН. Неиспользуемые настройки можно удалить из списка.

При наличии сохраненных настроек, рекомендуется заполнять реквизиты новой ТТН в следующей последовательности:

  • Выбрать параметры доставки;
  • Внести ручные корректировки (при необходимости).

После заполнения всех необходимых данных для печати товарно-транспортной накладной надо нажать на кнопку Печать.
При этом значения всех реквизитов обработки будут сохранены в привязке к документу-источнику. При повторном формировании ТТН для реализации или перемещения (или при изменении реквизита По документу в уже открытой обработке) значения всех реквизитов автоматически восстановятся.
