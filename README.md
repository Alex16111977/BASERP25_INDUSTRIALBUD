# BASERP25 - Industrial Building Business Automation

## Описание проекта

**BASERP25** (Business Automation Software for Enterprise Resource Planning) - это конфигурация системы **1C:Enterprise 8.3**, предназначенная для автоматизации бизнес-процессов в области промышленного строительства и управления ресурсами предприятия.

## Основные возможности

- **Учет товаров и материалов**: управление номенклатурой, складами, движением запасов
- **Финансовый учет**: бухгалтерский, налоговый и управленческий учет
- **Управление персоналом**: расчет заработной платы, учет рабочего времени, кадровый учет
- **Производство**: планирование, учет незавершенного производства, калькуляция себестоимости
- **Взаиморасчеты**: работа с контрагентами, управление дебиторской и кредиторской задолженностью
- **Денежные средства**: учет движения денежных средств, банковские операции, касса
- **Основные средства и НМА**: учет амортизации, ввод в эксплуатацию, выбытие

## Установка и развертывание

### Требования
- 1C:Enterprise 8.3 (платформа версии 8.3.20 или выше)
- Сервер СУБД (MS SQL Server / PostgreSQL / встроенная СУБД)

### Загрузка конфигурации

1. Создайте новую информационную базу в 1C:Enterprise
2. Откройте конфигуратор
3. Выполните команду **Конфигурация** → **Загрузить конфигурацию из файла**
4. Выберите файл `Configuration.xml`
5. Обновите конфигурацию базы данных

### Альтернативный способ (из XML-файлов)

1. Создайте новую информационную базу
2. В конфигураторе выполните **Конфигурация** → **Сравнить, объединить с конфигурацией из файлов**
3. Укажите путь к папке с XML-файлами
4. Примите изменения и обновите конфигурацию БД

## Разработка

### Система контроля версий
Проект использует Git для контроля версий. Конфигурация выгружена в формате XML (иерархическая структура) для удобства отслеживания изменений.

### Ветвление
- `main` - основная ветка (production-ready)
- `develop` - ветка разработки
- `feature/*` - ветки для новых функций
- `hotfix/*` - ветки для срочных исправлений
