РМК і обладнання

Використовуючи цю панель можна налаштувати та підключити торговельне обладнання: сканери штрих-кодів, термінали збору даних, Контрольно-касові машини (ККМ), що працюють у різних режимах.

Використовувати обладнання для підключення. Можливість використання зовнішнього торговельного обладнання: каси ККМ, сканери штрих-кодів, термінали збору даних, ваги тощо. Використовуючи гіперпосилання Обладнання для підключення, можна зробити настройку торговельного обладнання, що підключається. Підключення торговельного обладнання налаштовується для кожного робочого місця, де воно використовуватиметься.
Обмін з обладнанням для підключенняься OFFLINE. Можливість роботи з касами ККМ і вагами в режимі OFFLINE. Використовуючи гіперпосилання Правила обміну з обладнанням для підключення OFFLINE можна створити правила вивантаження даних про товари в каси ККМ або ваги, що працюють в режимі OFFLINE.
Шаблони магнітних карт. Використовуючи це гіперпосилання можна настроїти ті шаблони, які будуть застосовуватися при зчитуванні рідером магнітних дисконтних карт (карт лояльності).
Настройки РМК Використовуючи це гіперпосилання можна здійснити настройку списку торговельного обладнання, яке буде задіяне для робочих місць касирів.
Настройка РМК для поточного робочого місця призначене для виконання настройки каси, гарячих клавіш, автентифікації для цього робочого місця.
