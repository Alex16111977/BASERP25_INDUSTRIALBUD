Бюджетування та планування

Бюджетування

Планування

Бюджетування
  • Бюджетування. Функціональна опція визначає, чи використовується підсистема бюджетування для планування та аналізу господарських операцій.
  • Бюджетний процес. Функціональна опція визначає, чи використовується бюджетний регламент та чи проводиться виконання бюджетних завдань за ним.
  • Використовувати сповіщення для задач бюджетування. Визначає необхідність використання бюджетного процесу. Гіперпосилання Настроїти розклад дозволяє настроїти розклад для сповіщення по задачам бюджетування.
  • Автоматично запускати фонове завдання розрахунку збережених даних при проведенні документів. Записує дані в спеціальний регістр для прискорення формування бюджетних звітів.
  • Автоматично запускати фонове завдання розрахунку збережених даних бюджетування зі звітів. Записує дані в спеціальний регістр для прискорення формування бюджетних звітів.

Планування
  • Сезонні коефіцієнти. Призначені для можливості класифікацій номенклатури за різними групами з метою подальшого автоматичного коригування планів продажів з урахуванням фактора сезонності.
  • Плани продажу/збирання (розбирання)/купівель/виробництва. Призначені для прогнозного планування відповідних планів з урахуванням наявних залишків та інших статистичних даних. При цьому на підставі затверджених планів купівель/виробництва планів можуть бути автоматично сформовані відповідні замовлення постачальникам та замовлення на виробництво.
