Вивантаження та завантаження даних EnterpriseData

Обробка призначена для вивантаження і завантаження даних з використанням універсального формату EnterpriseData. Призначена для використання в інформаційних базах, що підтримують синхронізацію даних через універсальний формат.

Робота обробки, вбудованої в конфігурацію

Обробка, що поставляється в складі конфігурації, працює тільки в режимі завантаження даних.

Для завантаження даних необхідно вказати шлях до файлу, що містить дані для завантаження, і натиснути кнопку Виконати.

Для вивантаження даних, а також використання додаткових можливостей слід використовувати зовнішню версію даної обробки, що входить до складу поставки.

Спрощений режим роботи

Для вивантаження даних необхідно:

  1. Вказати Шлях до файлу вивантаження, в який будуть вивантажені дані
  2. Вказати Версію формату, в якій необхідно вивантажити дані
  3. Настроїти склад вивантажуваних даних:
    1. Вибрати об'єкти метаданих
    2. При необхідності вказати відбір за періодом для документів
    3. При необхідності настроїти довільні відбори для довідників і документів
  4. Виконані настройки відборів можна зберегти в файл і використовувати при наступних сеансах роботи (кнопки Зберегти настройкиі Відновити настройки).

 

Для завантаження даних необхідно вказати шлях до файлу, що містить дані для завантаження

Режим розробника

Режим можна включити за допомогою кнопки Ще. Надає наступні додаткові можливості:

  1. Вивантаження і завантаження даних з тексту. В цьому випадку в якості джерела і одержувача даних служить не файл, а текстове поле розташоване на формі. Цей режим зручний, якщо даних небагато, і результат вивантаження зручно перенести в обробку завантаження через буфер обміну.
  2. Використання зовнішніх модулів обміну через універсальний формат. Для цих цілей слід вказати шлях до зовнішньої обробки, в модулі якої розташований зовнішній модуль менеджера обміну.
  3. Вивантаження даних, зареєстрованих на вузлі плану обміну. Для цих цілей слід вказати Вузол обміну при вивантаженні. В цьому випадку настройки складу вивантажуваних даних і настройка відбору недоступні.
