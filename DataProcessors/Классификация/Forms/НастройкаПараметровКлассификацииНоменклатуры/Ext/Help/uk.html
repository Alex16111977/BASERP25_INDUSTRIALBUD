Встановлення параметрів для проведення ABC/ XYZ класифікації товарів

Для проведення ABC/ XYZ класифікації товарів необхідно вказати наступні параметри:

  • Встановити прапори у тих показників, відповідно до значень яких буде проводитися ABC/ XYZ класифікації товарів (виторг, валовий прибуток, кількість проданих товарів).
  • Зазначити період і кількість періодів, за які будуть аналізуватися дані, для ABC класифікації.
  • Зазначити період і кількість періодів, за які будуть аналізуватися дані, для XYZ класифікації. Для XYZ-класифікації додатково потрібно зазначити підперіоди, які будуть порівнюватися між собою в зазначеному періоді.

Настройка ABC/ XYZ класифікації товарів на розподільних центрах

Для проведення ABC/XYZ-класифікації товарів на розподільних центрах необхідно в настройках класифікації встановити прапорці Розраховувати продажі для розподільних центрів. При розрахунку будуть враховуватися продажі з тих складів, для яких даний розподільний центр (склад) встановлено в якості джерела переміщення для основного транспортного обмеження.
