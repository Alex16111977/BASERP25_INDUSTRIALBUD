Розміщення номенклатури по комірках (довідково)

Форма призначена для зазначення адрес складських комірок, в яких зберігається товар. Ця можливість надається в тому випадку, якщо на складі або в приміщенні комірки використовуються довідково. Для кожного товару можна вказати одну основну і довільну кількість додаткових комірок. Інформація використовується при друку завдань на розміщення товарів.

При оформленні надходження товарів на склад форма може бути заповнена на підставі документів: Придбання товарів і послуг, Надходження товарів на склад, Повернення товарів від клієнта, Прибутковий ордер на товари, Приймання товарів на зберігання. У цьому випадку таблична частина заповнюється автоматично тими товарами з документів поставки, для яких назначені комірки, необхідні для групування даних при друці завдань на розміщення товарів.

Довідкове зазначення інформації про комірки

Друк завдання на розміщення товарів (якщо комірки на складі використовуються довідково)

Довідкове зазначення інформації про комірки
  • У формі Розміщення номенклатури по комірках (довідково) при відкритті автоматично заповнюється поле Склад, на який приймається товар і поле Документ приймання, яким фіксується приймання товарів на склад.
  • У табличній частині форми буде автоматично заповнена інформація про товари, згідно з документом приймання. Якщо для товару вже була визначена комірка, то інформація про неї буде заповнена в табличній частині. Для кожного товару можна визначити основну і додаткову комірку. Для товару можна призначити будь-яку кількість комірок (кнопка Додати). Використовуючи кнопки Зробити основною і Зробити додатковою можна змінювати властивості комірки: додаткову комірку призначити основною і навпаки.
  • Після зазначення комірок для всіх товарів натисніть на кнопку Записати для запису інформації для подальшого її використання.

Друк завдання на розміщення товарів (якщо комірки на складі використовуються довідково)
  • У формі Розміщення номенклатури по комірках (довідково) натисніть на кнопку Друк завдання на розміщення товарів.
  • Буде роздруковано завдання на розміщення товарів.
  • У завданні будуть показані основні і додаткові комірки.
  • Список основних комірок буде відсортовано відповідно до тієї структури, яка вказана для складських комірок.
  • Якщо для комірок визначені робочі ділянки, то список комірок буде розділений по робочих ділянках.

Для кожного товару можна визначити основну і додаткову комірку. Для товару можна призначити будь-яку кількість комірок (кнопка Додати). Використовуючи кнопки Зробити основною і Зробити додатковою можна змінювати властивості комірки: додаткову комірку призначити основною і навпаки.
