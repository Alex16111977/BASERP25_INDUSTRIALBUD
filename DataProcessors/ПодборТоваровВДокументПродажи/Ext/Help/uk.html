Підбір товарів в документ продажу

Робоче місце призначене для підбору товарів в табличну частину документа. В диалоговом окне подбора можно установить различные фильтры для ограничения области поиска товаров: по иерархии номенклатуры, по виду номенклатуры, по товарам другого качества. Відповідно до обраного варіанту відбору буде показаний список груп номенклатури, список видів номенклатури або поле для вибору товару вихідної якості.

В левой части диалогового окна отображается перечень товаров, принадлежащих указанной в правой части группе номенклатуры, виду или товару другого качества.
Товары, для которых есть товары другого качества, отмечены в списке коричневой черточкой.

Особливості робочого місця:

  • При використанні фільтра за товарами іншої якості в полі товар можна вибрати номенклатуру, для якої ведуться товари з якістю відмінним від нового. Також встановити фільтр і вихідний товар можна по кнопці контекстного меню Товари іншої якості.
  • При підборі товарів можна використовувати швидкий відбір за видом і властивостями номенклатури. Список параметрів, які використовуються для швидкого відбору налаштовується для кожного виду номенклатури. Для реализации такого отбора необходимо установить фильтр по: виду и свойствам и выбрать нужный вид номенклатуры. У діалоговому вікні буде показаний список тих параметрів, які вказані в настройках швидкого відбору для обраного виду номенклатури. Для відбору потрібного товару потрібно встановити прапорці для потрібних значень параметрів.
  • Для кожного товару показуються ціни по тому виду ціни, який вказаний в документі і залишки товарів на тому складі, який вказаний в документі. Для перегляду інших цін по товару використовуйте кнопку Ціни.
  • Предусмотрена возможность установки отбора по цене, задав диапазон цены от и до.
  • Для підбору товарів які є в наявності потрібно активізувати прапор Тільки в наявності на складі з документа.
  • Для установки отбора по сегменту номенклатуры выберите сегмент из списка (подбор по сегменту возможен только при включенной функциональной опции Сегменты номенклатуры в разделе НСИ и администрирование - CRM и маркетинг - Маркетинг).


Гарячі клавіші при підборі товарів:

  • Alt + 1-перехід до строки пошуку.
  • Alt+2 – перейти до фільтрів.
  • Alt+3 – перейти до списку товарів.
  • Alt+4 – перейти до списку підібраних товарів.

Переходи також можна здійснювати за допомогою клавіші Tab.
