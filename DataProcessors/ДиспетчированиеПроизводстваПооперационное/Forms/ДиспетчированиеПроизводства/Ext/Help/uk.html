Створення маршрутних листів


Для створення маршрутних листів використовується команда Сформувати маршрутні листи.
Команда стає доступною, якщо є необхідність у створенні.

Маршрутні листи створюються на кожен етап виробництва. Кількість маршрутних листів відповідних одному етапу визначається в настройці етапу Розбивати маршрутні листи. При необхідності маршрутний лист можна розбити на кілька за допомогою команди Розбити маршрутний лист.

Планування розкладу (доступно тільки для методики Поопераційне планування)


Для планування розкладу використовується команда Планувати розклад, яка відкриває спеціальне робоче місце Поопераційне планування.

Оформлення пов'язаних документів


При виконанні маршрутних листів необхідно оформити наступні документи:

  • Виробіток співробітників - оформляється при виконанні маршрутного листа, якщо вказано які бригади виконують роботи;
  • Випуск продукції та виконання робіт - оформляється після виконання маршрутного листа.

Якщо на підставі маршрутного листа потрібно оформити зазначені документи, то у відповідній колонці буде виведений індикатор.
Створити необхідний документ можна клацанням на індикаторі або за допомогою команди розташованої в підменю Задачі.

Індикатори


Для маршрутних листів виводяться різні індикатори, які дозволяють визначити наявність проблем або показують довідкову інформацію.

  • Попередження-для розшифровки тексту попередження необхідно виконати подвійний клік на індикаторі;
  • Не сформовано розклад (доступно тільки для методики Поопераційне планування);
  • Попередні етапи не завершені:
  • Потрібно оформити виробітку співробітників;
  • Потрібно оформити випуск продукції або виконання робіт.

Перегляд розкладу (доступно тільки для методики Поопераційне планування)


Для перегляду поопераційного розкладу використовується команда Розклад. Розклад може бути відображено у вигляді списку або у вигляді діаграми Ганта.
Є можливість індикації розкладу виділених маршрутних листів в одному з двох режимів:

  • Режим відбору – відображається розклад тільки виділених документів;
  • Режим підсвічування – для індикації розкладу виділених маршрутних листів використовується колірна підсвітка.

Друк змінно-добових завдань


Для друку змінно-добових завдань використовується команда Друк змінно-добових завдань. Для виконання друку в маршрутних листах повинні бути призначені бригади за видами робіт.
