Призначений для роботи з файлами, приєднаними до документа або елементу довідника. Файли зберігаються в BAS Документообігу.

Відкривається по команді Приєднані файли панелі навігації.

У списку виводяться:

  • - наявність ЕП або шифрування.
  • Найменування- повне ім'я файлу. Значок поруч з найменуванням вказує тип файлу. При наявності в колонці виводиться також і опис - докладні відомості про файл.
  • Автор- користувач, який створив файл;
  • Дата зміни-дата зміни даних файлу (місцевий час);

За допомогою команди Ще - Додатково - Змінити форму можна додати відомості:

  • Дата створення - дата і час введення файлу в програму;
  • Редагує- користувач, який зайняв файл для редагування, тощо.

Додавання нового файлу

  • Новий файл можна Додати з диска.

  • За допомогою команди Ще - додати друковану форму можна додати як файл друковану форму власника

Перегляд картки файлу

  • Натисніть Ще - відкрити картку (натисніть клавішу F2), для того щоб переглянути властивості виділеного в списку файлу.

Редагування приєднаних файлів

  • Для редагування файлів на комп'ютері повинні бути встановлені відповідні прикладні програми, наприклад, для редагування файлів у форматі Microsoft Word має бути встановлена ця прикладна програма.
  • Команда Переглянути відкриває файл зовнішнім додатком для перегляду або редагування- в залежності від того, позначений чи ні файл як зайнятий користувачем для редагування.
  • команда Редагувати відкриває файл в режимі редагування, а також виконує захоплення файлу в інформаційній базі. Захоплений файл буде недоступний для редагування іншим користувачам системи і позначається в списку зеленим кольором.
  • Після закінчення редагування збережіть файл, закрийте зовнішню прикладну програму. Натисніть Закінчити редагування, ця дія звільняє файл, зберігає його в програмі і знімає позначку зайнятості для редагування.
  • Команда Ще - Скасувати редагування звільняє файл, не зберігаючи виконані в ньому зміни в програмі, і знімає позначку зайнятості для редагування. Команда може бути корисна, якщо необхідно відмовитися від результатів редагування файлу (наприклад, при редагуванні в файл були внесені небажані зміни або помилки).

Збереження файлу

  • За допомогою команди Ще - Зберегти як можна зберегти файл в зазначений каталог на жорсткому або мережевому диску.
  • Також можна скористатися відповідною командою контекстного меню по правій кнопці миші.
  • Відповідною командою меню Ще або контекстного меню по правій кнопці миші можна Відкрити каталог файлу в Провіднику Windows.

Оновлення файлу

  • З допомогою команди Ще - Оновити з файлу на диску можна оновити файл з оригінального файлу на диску (з якого спочатку був створений файл у програмі).
  • Також можна скористатися відповідною командою контекстного меню по правій кнопці миші.

Електронний підпис

У випадаючому меню по кнопці Електронний підпис доступні команди для роботи з електронним підписом файлів:

  • Підписати- створює ЕП для файлу. Перед створенням ЕП програма пропонує вибрати сертифікат ЕП з наявних на комп'ютері. Вибір сертифіката можна зробити і під час персональних настройок програми. Підписаний об'єкт стає недоступним для редагування і відображається в списках зі значком.
  • Зберегти разом з електронним підписом - зберігає підпис у файл на диску. Команду можна використовувати, наприклад, якщо необхідно передати підписаний файл іншій особі.
  • Додати електронний підпис з файлу - додає підпис до файлу. Команду можна використовувати, наприклад, якщо підпис (файл підпису) був отриманий пізніше, ніж підписаний файл.
