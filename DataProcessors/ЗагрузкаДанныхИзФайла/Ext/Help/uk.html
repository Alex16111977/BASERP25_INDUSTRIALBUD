Надає можливість завантаження табличних даних з файлів в довільні списки і табличні частини документів. Наприклад, для разового перенесення нормативно-довідкової інформації при переході з інших програм, або для регулярного завантаження даних у табличні частини при заповненні документів.

Переваги завантаження даних з файлу

  • Дозволяє задіяти для введення даних можливості інших програм (наприклад, можна вводити дані в Excel, а потім завантажити їх в програму);
  • Не потребує для введення даних у файл наявності на комп'ютері програми BAF, тобто введення даних в такий файл може здійснюватися у віддалених офісах, невеликих складах, точках роздрібних продажів, тобто там, де може не бути можливості налагодити повноцінну роботу програми BAF (наприклад, комп'ютер вже застарів для роботи з даною програмою);
  • Дозволяє підключити до введення даних додаткові кадри. Введення даних у зовнішній файл вимагає лише дотримання структури таблиці, а також знання правил введення в тій програмі, для якої він призначений, наприклад, Excel, тому може бути доручено співробітникам, які мають лише початкові навички роботи з комп'ютером.
