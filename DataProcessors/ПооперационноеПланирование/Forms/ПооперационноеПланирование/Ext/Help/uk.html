Поопераційне планування

Робоче місце призначене для формування поопераційного розкладу виробництва.
При плануванні виконується розміщення операцій маршрутних листів, що належать підрозділам з поопераційною методикою управління. Операції розміщуються в розкладі з дати початку виконання маршрутного листа.

Розрахунок розкладу


Для розрахунку розкладу використовується команда Розрахувати, яка відкриває форму настройки параметрів розрахунку.

Запис розкладу


Для запису побудованого розкладу служить команда Записати.

Якщо в параметрах розрахунку розкладу вибрано кілька моделей планування, то при розрахунку буде побудовано декілька варіантів розкладу, за кількістю обраних моделей. Для запису доступний лише один з побудованих варіантів розкладу – оптимальна модель. Для вибору оптимальної моделі слід виконати подвійний клік на відповідному елементі діаграми і вибрати команду Встановити в якості оптимальної моделі. Оптимальна модель відображається в розкладі жирним шрифтом.

У параметрах розрахунку розкладу може бути вказаний сценарій поопераційного планування. Розклад, побудований за допомогою сценарію, призначений виключно для аналізу ефективності даного сценарію, і не доступний для запису.

Аналіз розкладу


Для аналізу розрахованого розкладу до його запису служить команда Аналіз розкладу, яка відкриває звіт Аналіз поопераційного розкладу.
