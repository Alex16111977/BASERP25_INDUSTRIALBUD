Оформлення виробництва без замовлень

Робоче місце призначене для автоматичного і ручного оформлення виробництва без замовлення і розподілу зворотних відходів.
Для робочого місця доступна ннучка настройка фільтрів за організацією, підрозділом тощо.

Продукція

У списку відображається інформація:

  • про випущену продукцію, на яку необхідно віднести витрати;
  • про настройки моделі формування вартості продукції.

Команди:

  • Оформити за настройками - призначена для автоматичного формування документів для виділених рядків, відповідно до настройок моделі формування вартості;
  • Оформити вручну - Виробництво без замовлення - призначена для пакетного формування документів Виробництво без замовлення для виділених рядків;
  • Оформити вручну - Розподіл зворотних відходів - призначена для пакетного формування документів Розподіл зворотних відходів для виділених рядків;
  • Настройка моделі формування вартості - Настроїти - призначена для пакетної настройки моделі формування вартості для виділених рядків за допомогою форми настройки моделі формування вартості;
  • Настройка моделі формування вартості - Очистити - призначена для пакетної очистки настройок моделі формування вартості для виділених рядків;
