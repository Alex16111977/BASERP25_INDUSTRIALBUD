Настройка контролю забезпечення

Настройка визначає винятки для контролю забезпечення номенклатури.

Можливі два варіанти настройки виключень контролю забезпечення:

  • При включеній настройці складу Контролювати забезпечення, для номенклатури, зазначеної в настройках, не контролюється забезпечення.
  • При вимкненій настройці складу Контролювати забезпечення, для номенклатури, зазначеної в настройках, буде контролюватися забезпечення.
