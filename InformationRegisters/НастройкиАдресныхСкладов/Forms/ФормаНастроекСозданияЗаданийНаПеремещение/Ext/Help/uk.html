Правила створення завдань

Правила визначають умови пошуку товарів і умови підбору комірок для розміщення. Створення завдань може відбуватися за одним з правил:

  • Підживлення зон швидкого відбору
  • Переміщення за правилами розміщення відокремлених товарів.

Варіанти розбиття по робочих ділянках
  • З розбиттям по робочих ділянках - для кожної робочої ділянки складу будуть створюватися окремі завдання.
  • По одній робочій ділянці - пошук товарів для переміщення відбувається тільки на обраній робочій ділянці.
  • Без розбиття по робочих ділянках - всі знайдені товари переміщаються одним завданням.

Завдання створюються з обмеженнями за вагою і об'ємом. У кожне завдання буде додана мінімум одна упаковка, навіть якщо вона перевищує обмеження.

У створених завданнях комірки сортуються по робочій ділянці, потім за порядком обходу, потім за кодом комірки.

Робота регламентного завдання

Вкажіть найменування і розклад регламентного завдання. Розклад починає діяти відразу після збереження настройок адресних складів.

Подробиці роботи регламентного завдання записуються в журнал реєстрації за подією "Автоматичне створення завдань на переміщення".
