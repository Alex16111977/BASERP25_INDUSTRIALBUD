Примірники бюджетів

Документ призначений для введення планових значень за обраним Видом бюджету за певний період в розрізі Сценарію, а так само Організації і Підрозділу, якщо вони налаштовані в Моделі бюджетування.

Документ є основним інструментом для встановлення значень за Статтями бюджетів з деталізацією до аналітики. Документ використовується для первинного введення планових даних по бюджетах і для коригувань.

Дата документа, введені плани (або зміни планів) використовуються в бюджетних звітах, дозволяючи отримувати різні версії бюджетів на дату актуальності звіту.

У списку передбачені відбори:

  • Період- зазначення тимчасового періоду для вибору діючих у ньому бюджетів
  • Вид бюджету- відбір бюджетів певного виду
  • Сценарій- відбір за сценарієм планування
  • Статус- обмеження списку відображуваних бюджетів за статусом
  • Відповідальний- відбір по співробітнику, який готував бюджети


За допомогою команди Встановити статус можна присвоїти статуси відразу декільком документам.

Якщо для виду бюджету встановлені додаткові аналітичні розрізи, то при відборі за даним видом бюджетів в список примірників бюджету будуть додані додаткові поля зі значеннями аналітичних розрізів, які використовуються.

Приклад Для виду бюджету встановлено додатковий аналітичний розріз Контрагенти. У списку примірників бюджетів при відборі за даним видом бюджетів додасться поле Контрагенти. При виведенні загального списку примірників бюджетів за іншими видами- поле буде автоматично приховано.

Також можна налаштувати список видів бюджету, які виводяться завжди в будь-якому випадку. Настройка списку аналітик бюджетної форми проводиться: Ще - Аналітики відображувані завжди. У діалоговому вікні необхідно вказати список аналітичних вимірів.
