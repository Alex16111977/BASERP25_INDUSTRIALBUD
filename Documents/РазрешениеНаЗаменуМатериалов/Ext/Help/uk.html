Дозвіл на заміну матеріалів

Документ Дозвіл на заміну матеріалів призначений для реєстрації можливості заміни матеріалу (або набору матеріалів) на аналогічний.

Застосування

Дозвіл застосовується як для ручного підбору аналогів матеріалів в документах, так і для автоматичної заміни на аналоги при запуску замовлення у виробництво (формуванні етапів), при забезпеченні виробництва або при замовленні матеріалів у виробництво.

Життєвий цикл дозволу на заміну

Для управління життєвим циклом дозволу на заміну використовується статус. Можливі статуси:

  • Створено - чернетка дозволу.
  • Затверджено - чинний дозвіл. При переведенні в цей статус дата початку періоду дії дозволу заповнюється поточною датою, якщо раніше не була заповнена.
  • Припинено - дозвіл, дія якого припинено. При перекладі в цей статус дата закінчення періоду дії заповнюється поточною датою.

Дія дозволу на заміну

За замовчуванням дозвіл на заміну задається з дією По підприємству в цілому. Але область дії може бути обмежена за такими параметрами:

  • Напрям діяльності
  • Підрозділ
  • Виріб, що виробляється
  • Ресурсна специфікація
  • Замовлення на виробництво

Якщо дозвіл на заміну матеріалів задається для ресурсної специфікації, то набір замінних матеріалів визначаться строками табличної частини Матеріали та роботи даної специфікації.

При наявності декількох діючих аналогів для здійснення автоматичної заміни вибирається найбільш точно відповідний дозвіл, з більш вузько заданою дією. Пріоритет дії дозволів визначається наявністю уточнення області дії.
Наприклад, якщо є дозвіл на заміну, що діє по підприємству в цілому, і є відповідний дозвіл для конкретного напряму діяльності, то буде вибрано дозвіл з уточненим напрямом діяльності.
Якщо є кілька дозволів з уточненням дії за різними параметром, то більш пріоритетним вважається дозвіл, у якого уточнено параметр, розташований вище в списку:

  • Напрям діяльності
  • Підрозділ
  • Виріб
  • Замовлення

Якщо для номенклатури створено кілька дозволів з однаковими областями дії, тоді застосовується найсвіжіший дозвіл - з більш пізньою датою початку дії.
