Документ "Розрахунковий документ"

Допоміжний документ, який не робить ніяких проводок, призначений для ручного ведення обліку за розрахунковими документами, тобто для організації субконто "Партія" на рахунках розрахунків з контрагентами. Реквізити необхідні для заповнення:

  • Організація
  • Контрагент
  • Договір
