Реєстрація інших доходів

Документ призначений для реєстрації виплат різних доходів поза рамками трудових відносин. Він доступний, якщо в настройках розрахунку зарплати обрана відповідна можливість.

Реєстрація виплат цим документом проводиться виключно з ціллю обчислення ПДФО та/або страхових внесків та відображення їх у відповідній звітності. Також ці суми враховуються при формованих даних для відображення зарплати в бух. обліку.

За зареєстрованим сумами не утворюється заборгованість організації перед одержувачами таких доходів, тобто вони не враховуються при заповненні відомостей на виплату зарплати, ні при формуванні аналітичних звітів по зарплаті.
Передбачається, що всі грошові розрахунки поза рамками трудових відносин проводяться в бухгалтерській програмі (підсистемі).
Якщо потрібно виконувати розрахунки з співробітниками по іншим доходам так само, як розрахунки з оплати праці, то їх можна реєструвати, наприклад, як довільне разове нарахування.

Див. також
- Для реєстрації договорів цивільно-правового характеру на виконання робіт або надання послуг, а також договорів авторського замовлення призначені окремі документи;
- Для реєстрації соціального посібника на поховання працівника, який виплачується його родичу (за рахунок соціального страхування), призначений документ Одноразова допомога за рахунок соціального страхування;
- Зареєстровані документом виплати можна проаналізувати з допомогою звіту Незарплатные доходи.
