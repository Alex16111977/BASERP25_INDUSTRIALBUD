Настройка


Для работы с ТН необходимо включить функциональную опцию Товарно-транспортные накладные в разделе НСИ и администрирование – Продажи - Печать документов. 

Документы-основания


Создание ТН предусмотрено для следующих документов: Перемещение товаров, Реализация товаров и услуг, Передача товаров хранителю, Возврат товаров поставщику, Возврат сырья давальцу, Передача давальцу, Передача сырья переработчику, Отгрузка товаров с хранения. Для документов Передача давальцу и Передача сырья переработчику ТТН добавляются из Задания на перевозку

Основное

На закладке Основное заполняются значения полей, которые находятся в шапке печатной формы 1-ТН.

Товарный раздел

На закладке Товарный раздел заполняются документы, на основании которых происходит формирование печатных форм 1-ТН и транспортной накладной.
Есть возможность настройки вывода информации о товарах в товарном разделе печатной формы 1-ТН:

  • Номера накладных - выводится список накладных, на основании которых формируется печатная форма. Настройка доступна всегда.
  • Товарный состав - выводится полный список товаров из накладной. Настройка доступна для выбора тогда, если Транспортная накладная формируется по одному документу-основанию.

    Определяется Масса (брутто) груза. Для автоматического расчета Массы брутnо должен быть указан Вес товаров в справочнике Номенклатура или Вес (брутто) упаковок товаров. Определяются     правила вывода массы брутто груза прописью в заголовок печатной формы: целое число (округление в большую сторону) или точное значение.

Транспортный раздел

На закладке Транспортный раздел заполняются данные о сроках, адресах и исполнителях перевозки.

Сведения о транспортном средстве и водителе заполняются автоматически при вводе ТТН из Задания на перевозку. При вводе ТТН из накладной сведения о транспортном средстве и водителе заполняются вручную или путем подбора из введенных ранее Транспортных накладных. Для подбора нужно начать заполнять реквизиты ФИО и код водителя и Гос. номер транспортного средства.

Масса по техпаспорту транспортного средства не выводится на печать и необязательна для заполнения; используется для расчета Общей массы с грузом.

Общая масса с грузом рассчитывается как Масса брутто груза + Масса по техпаспорту транспортного средства и может корректироваться вручную. 

Формирование документа


Транспортная накладная формируется из накладной при нажатии на кнопку Печать - Товарно-транспортная накладная. В документе можно указать, в каком виде будут выводиться данные в табличной части - как номера накладных или как товарный состав.

Транспортная накладная может быть сформирована также из формы списка или из табличной части документа Задание на перевозку по кнопке-пиктограмме Оформить транспортные накладные.

Если в документах-основаниях указаны одинаковые значения реквизитов и настроек - Организация, Грузополучатель, Плательщик, Адрес доставки, Перевозчик (если используется) - то для таких документов автоматически сформируется одна общая транспортная накладная.

Дополнительно транспортная накладная может быть сформирована из обработки Помощник продаж по кнопке Транспортная накладная на странице Документы.

Предусмотрена возможность автоматического формирования транспортной накладной. Для этого в настройке Помощника продаж необходимо установить флаг Создавать транспортную накладную по умолчанию в группе команд Все действия - Настройка...

  • Реализация товаров и услуг, 
  • Перемещение товаров, 
  • Возврат товаров поставщику,
  • Возврат сырья давальцу.

Групповое объединение накладных


Предусмотрена возможность объединения нескольких документов-оснований в одну транспортную накладную. Для этого необходимо из табличной части документа нажать кнопку Выбрать - в списке отобразятся ранее созданные транспортные накладные с совпадающими значениями реквизитов Организация, Грузополучатель, Плательщик, Адрес доставки, Перевозчик (если используется) - и указать необходимый документ.

Обязательное условие данного способа создания транспортной накладной заключается в выборе вариантов доставки Самовывоз и Силами перевозчика с нашего склада. Если указаны варианты доставки До клиента и Силами перевозчика по адресу, то документ Транспортная накладная можно создать только из Задания на перевозку.

Печатные формы


Печатная форма Товарно-транспортная накладная доступна как из документов-оснований, так и из документа Транспортная накладная.
