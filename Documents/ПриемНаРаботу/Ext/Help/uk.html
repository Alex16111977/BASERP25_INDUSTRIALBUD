Прийом на роботу

Документ призначений для реєстрації факту укладення трудового договору зі співробітником і для оформлення відповідного наказу про прийняття за типовою формою П-1.

Див. також:

Див. також

- Для оформлення в програмі прийому на роботу всіх співробітників, які вже працюють на підприємстві на момент початку використання програми, призначений спеціальний документ Початкова штатна розстановка;
- Для оформлення прийому на роботу списку співробітників призначений документ Прийом на роботу списком.
