Принятие к учету ОС (2.1)

Документ предназначен для отражения в системе факта принятия к учету основных средств.

Проводки регламентированного учета в документе формируются при условии, что на счете капитализации затрат (субсчет 15 счета) есть суммы в разрезе подразделения, указанного в документе. Возможно потребуется расчет себестоимости в закрытии месяца, если затраты формируются внутренним потреблением номенклатуры.

Основные средства

Основное
  • Принятие к учету на основании - документ, на основании которого вводится принятие к учету объектов основных средств;
  • Подразделение - подразделение организации, в котором принимается к учету основное средство;
  • Событие - событие основного средства, которое отражается данным документом;
  • Счета учета - счета, на которых будут учитываться основные средства и накапливаться амортизация;
  • Местонахождение - место хранение основного средства и лицо, ответственное за его сохранность;
  • Направление деятельности - накладывает отбор на подбираемые в документ основные средства.


Амортизация


На закладке Амортизация уточняются следующие реквизиты по данным бухгалтерского и налогового учета:

  • Начисление амортизации - указывается признак наличия/отсутствия на текущий момент начисление амортизации.
  • Метод начисления амортизации - способ начисления амортизации по основным средствам.
  • Срок использования (в месяцах) - срок полезного использования основных средств.
  • График амортизации - график амортизации основных средств только в организациях с сезонным характером производства.
На закладке Отражение расходов указывается Статья и Аналитика расходов для начисления амортизации.

Основные средства


На закладке заполняется список принимаемых к учету основных средств.

При выборе основного средства инвентарный номер заполняется автоматически.
Для быстрого заполнения документа группой однотипных основных средств нужно ввести в табличное поле хотя бы один такой объект и автоматически заполнить табличное поле основными средствами с таким же наименованием по кнопке Заполнить - По наименованию.

При оформлении принятия к учету на основании инвентаризации - для каждого из основных средств потребуется указание стоимости.

При применении производственного метода начисления амортизации на закладке Основные средства в табличной части добавляются поля:

  • Показатель наработки - наименование характеристики, объем которой используется для начисления амортизации.
  • Предполагаемый объем наработки - предполагаемый объем продукции (работ), исходя из которого будет начисляться амортизация. Объем указывается в единицах измерения параметра выработки.
