Початкові залишки НЗП по партіях виробництва

Документ призначений для введення залишків незавершеного виробництва по партіях, виконання робіт по яких почалося до і триває після включення в інформаційній базі обліку собівартості.
Документ використовується, коли у зовнішній системі ведеться облік кількісних і вартісних залишків НЗВ з деталізацією до сутностей ERP (підрозділів, статей калькуляції, партій виробництва, матеріалів/напівфабрикатів, видів робіт, статей витрат).
Документ формується по організації і підрозділу. У документі на відповідних закладках вносяться залишки матеріальних витрат, трудовитрат та інших витрат по партіях виробництва.

Матеріали і роботи

Таблична частина призначена для фіксації матеріальних залишків НЗВ, деталізованих по партіях виробництва.
Для роботи з табличною частиною передбачені наступні команди:

  • Команда Завантажити з файлу... - призначена для заповнення табличної частини, за допомогою виклику стандартного діалогу завантаження даних табличних документів.
  • Команда Заповнити статтю калькуляції - призначена для заповнення статті калькуляції у виділених рядках табличної частини.

У міру заповнення документа, в заголовку закладки, показується інформація про кількість рядків табличної частини.
