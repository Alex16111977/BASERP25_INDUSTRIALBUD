Вхідна довідка про заробіток для розрахунку допомоги

Документ призначений для реєстрації в програмі даних з довідки про заробіток для розрахунку допомог, наданої співробітником відповідно до законодавства. Зареєстровані документом дані можуть враховуватися при розрахунку середнього заробітку для допомог соціального страхування.
