Больничный лист

Документ предназначен для регистрации факта предъявления сотрудником листка нетрудоспособности, как с целью назначения пособия, так и для отражения его отсутствия.
Также документ позволяет рассчитать и начислить: 
• доплату до полного среднего заработка за дни нетрудоспособности;
• зарплату за время до начала отпуска по беременности и родам.

При расчете страхового стажа могут использоваться либо данные стажа, введенные для физического лица в справочнике, либо данные справки ПФУ (документ "Ввод сведений для расчета страхового стажа"). 
Если данные стажа введены с помощью документа "Ввод сведений для расчета страхового стажа", то только они и учитываются при расчете стажа в документе. В таком случае на закладке "Оплата"  
для отображения страхового стажа выводится соответствующая надпись, при нажатии на которую можно открыть список документов ввода стажа. 
Для удобства проверки введенного по данным ПФУ стажа есть возможность формировать печатную форму подробного расчета стажа по данным ПФУ.
