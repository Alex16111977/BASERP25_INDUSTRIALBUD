Больничный лист

Документ предназначен для регистрации факта предъявления сотрудником листка нетрудоспособности, как с целью назначения пособия, так и для отражения его отсутствия.
Также документ позволяет рассчитать и начислить: 
• доплату до полного среднего заработка за дни нетрудоспособности;
• зарплату за время до начала отпуска по беременности и родам.

При расчете страхового стажа могут использоваться либо данные стажа, введенные для физического лица в справочнике, либо данные справки ПФУ (документ "Ввод сведений для расчета страхового стажа"). 
Если данные стажа введены с помощью документа "Ввод сведений для расчета страхового стажа", то только они и учитываются при расчете стажа в документе. В таком случае на закладке "Оплата"  
для отображения страхового стажа выводится соответствующая надпись, при нажатии на которую можно открыть список документов ввода стажа. 
Для удобства проверки введенного по данным ПФУ стажа есть возможность формировать печатную форму подробного расчета стажа по данным ПФУ.

Если не отработано полного календарного месяца с даты приема до начала события, расчет среднего заработка для оплаты больничного проводится по фактически отработанному времени. 
В таком случае для месяца начала события рассчитывается сумма ФОТ и отработанные календарные дни до даты начала больничного (исключая дни невыходов по уважительной причине). В расчет средней за этот месяц включается сумма, рассчитанная по формуле: ФОТ/(количество календарных дней в месяце)*(количество отработанных календарных дней). 
Если начало события приходится на месяц приема, то расчет в документе вернет нулевой средний заработок, тогда необходимо открыть форму-калькулятор (кнопка возле поля "Средний заработок") и нажать кнопку "Дополнить по фактически отработанному времени". 
Если начало события приходится на следующий после приема месяц, то за месяц приема зарплата уже должна быть обязательно начислена. За этот месяц средняя по фактически отработанному времени рассчитывается автоматически.
