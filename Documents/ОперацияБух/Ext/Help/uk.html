Операція (регламентований облік)

Документ призначений для відображення в регламентованому обліку проводок по господарських операціях, для яких у конфігурації не створено документів оперативного обліку.

При заповненні документа господарська операція відображається в ньому у вигляді бухгалтерської проводки за планом рахунків.

Документ призначений для виконання наступних операцій:

  • Введення кореспонденції рахунків з бухгалтерського та податкового обліку
  • Сторнування записів одного або декількох документів


Для сторнування документа необхідно вказати пов'язаний документ або вибрати список документів і натиснути кнопку Заповнити сторно. Сторновані проводки заповняться автоматично.

Для документа передбачена друкована форма Бухгалтерська довідка.
