Форма настройок на партії

Постатейну витрату може бути розподілено:

  • на партії - постатейна витрата розподіляється на партії виробництва. Можливий розподіл між партіями за правилом, або на перераховані вручну;
  • на інші статті витрат - можливість віднести постатейну витрату на статті витрат з іншим варіантом розподілу, наприклад з метою перекваліфікації витрати;
  • на НЗВ - якщо постатейну витрату не потрібно розподіляти в місяці розподілу, то можливо залишити витрату в незавершеному виробництві.

Залежно від того куди розподіляється постатейна витрата в документі змінюється склад закладок документа, дозволяючи вказувати детальні дані для розподілу. Для кожного напрямку розподілу потрібно вказати частку вартосты, за винятком якщо вибрано тільки один напрямок розподілу.

Правила розподілу

На закладці можна налаштувати правило розподілу, за яким будуть розподілятися витрати між підрозділами і партіями виробництва.
Розподіл виконується в 2 кроки:

  • постатейна витрата розподіляється між підрозділами;
  • на партії виробництва підрозділів.

Підрозділи, на які буде розподілена постатейна витрата, можна перерахувати вручну із зазначенням часток вартості на кожен підрозділ, або вибрати зі списку:

  • всі підрозділи;
  • поточне - підрозділ документа, вказаний на закладці Основне;
  • підпорядковані -підпорядковані підрозділи в довіднику Структура підприємства, підрозділи документа;
  • вищого рівня - підрозділи вищого рівня в довіднику Структура підприємства, підрозділи документа;
  • зазначені - дозволяє вказати перелік підрозділів вручну, без зазначення часток вартості.

Розподіл між партіями виробництва і підрозділами може бути виконано за загальним правилом. У цьому випадку підрозділи визначаються правилом розподілу між партіями з відповідним відбором по підрозділах. Для застосування такого варіанті потрібно вибрати варіант Розподіляти за єдиною базою для партій і підрозділів. Якщо ж потрібно суму витрати розподілити по підрозділах за одним правилом, а всередині підрозділів застосувати інше правило, то слід вибрати настройку Розподіляти за різними базами для партій і підрозділів.

Доступні бази розподілу між підрозділами:

  • задається вручну - на формі з'являється закладка Підрозділи;
  • за показником-вказується в додатковому полі Показник;
  • вартість матеріалів;
  • сума оплати праці;
  • Вартість матеріалів і сума оплати праці.

Доступні бази розподілу по партіях:

  • Вартість матеріалів і сума оплати праці.
  • вартість матеріалів;
  • сума оплати праці;
  • нормативна оплата праці;
  • кількість продукції з/без урахування майбутніх випусків;
  • кількість видів робіт;
  • кількість матеріалів;
  • вага продукції з/без урахування майбутніх випусків;
  • вага матеріалів;
  • об'єм продукції з/без урахування майбутніх випусків;
  • об'єм матеріалів;
  • планова вартість продукції з/без урахування майбутніх випусків.

Для баз розподілу по продукції можливий розподіл з урахуванням майбутніх випусків або без, тобто постатейна витрата буде включатися по мірі випуску продукції або буде повністю віднесена на випуск місяця розподілу.
Для інших баз розподілу постатейна витрата завжди включається у вартість продукції по мірі випуску продукції.

Партії виробництва

Містить інформацію про партії виробництва, на які потрібно віднести зазначену частку вартості розподілюваної суми витрат.
Додавання нових рядків виконується за допомогою форми підбору партій виробництва.

В якості партій виробництва виступають:

  • Етапи виробництва
  • Документи виробництва без замовлення
  • Звіти переробників

При цьому, для етапів виробництва доступні для вибору проміжні етапи, що входять в одну партію виробництва (партією є випусковий етап).

Інші статті витрат

Можливість віднесення витрат накопичених за розподіляється статті на статті з іншим варіантом розподілу. Дана можливість може бути використана для перекваліфікації витрат.

Зазначення частки вартості необхідно для попереднього розподілу витрат до моменту розрахунку фактичної собівартості витрати. Фактична собівартість витрати розраховується після виконання всіх операцій по закриттю місяця. Після розрахунку фактичної собівартості дані з розподілу витрат будуть уточнені.
