Зміна планових нарахувань

Документ призначений для відображення зміни умов оплати праці (планових нарахувань) декількох співробітників, у тому числі тимчасового.

Його введення передбачається, наприклад, для всіх співробітників, що займають певну позицію штатного розкладу, при зміні нарахувань для цієї позиції. У цьому разі документ можна ввести безпосередньо з картки позиції або документа, яким змінюються нарахування позиції (якщо ведеться історія зміни штатного розкладу).

Див. також
Для зміни нарахувань одного співробітника можна скористатися документами Зміна оплати праці або Кадрове переведення.

Якщо в організації виконується індексація, яка потім повинна впливати на розрахунок середнього заробітку (і вибрана відповідна можливість в настройках розрахунку зарплати), то її слід виконувати документом Індексація заробітку.

Заповнення документа

Таблиця співробітників заповнюється автоматично всіма співробітникам обраного підрозділу та/або обраної посади (позиції штатного розкладу, якщо воно ведеться).
Також співробітників можна додати (підібрати) в таблицю або вилучити з таблиці безпосередньо. По кнопці Заповнити можна перезаповнити список співробітників співробітниками цього підрозділу і/або цієї посади.

Безпосередньо в таблиці для перегляду і зміни доступні показники планових нарахувань обраних співробітників, що діють на зазначену дату зміни. Оскільки різні працівники можуть мати різні планові нарахування (за різними показниками), комірки з неактуальними для співробітника показниками (але актуальними для інших співробітників) для зміни недоступні.

Якщо потрібно змінити показники вже призначених співробітникам планових нарахувань, але і додати всім співробітникам нове нарахування або припинити чинне, то можна скористатися командами Додати нарахування всім співробітникам або Видалити нарахування у всіх співробітників в меню Ще. Вибране нарахування буде додано або припинено для всіх співробітників, заповнених в таблиці.

Крім того, натиснувши на посилання в колонці Нарахування можна переглянути або відредагувати нарахування та ФОП кожного співробітника індивідуально.

Якщо документ введений із документа затвердження або зміни штатного розкладу, то позиція заповнюється згідно з цим документом і для зміни недоступна.
